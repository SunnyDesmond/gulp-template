# product_board

#### 项目介绍
基于gulp的项目模板

#### 使用
1. npm/cnpm install 
2. npm run dev  //开发环境
3. npm run build //生产环境

#### 一些说明
1.已经将`js`文件夹中的所有js合并成 `all.min.js`
2.已经将`sass`文件夹中的所有css合并成 `all.min.css`
3.在开发环境中，支持热加载（不用刷新浏览器，只要保存文件，浏览器就能自动刷新）
4.开发环境预览的默认地址是`localhost:8080`
5.再生产环境中,对`src`下的`img`文件下的图片进行了压缩，减小图片的体积
